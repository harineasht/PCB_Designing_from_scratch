# 🚀 Arduino Nano Motor Driver PCB Design

## 📌 Project Overview

This project demonstrates the **design and development of a custom PCB from scratch** integrating an **Arduino Nano**, **TB6612FNG motor driver**, and **push buttons** for control input.

The goal of this project is to understand the **complete PCB design workflow** — from schematic creation to PCB layout and real-world implementation.

---

## 🎯 Objectives

* Design a **custom PCB** instead of using breadboards
* Interface **Arduino Nano with TB6612FNG motor driver**
* Use **push buttons for manual control inputs**
* Learn **PCB layout principles and routing techniques**

---

## 🧰 Components Used

* Arduino Nano
* TB6612FNG Motor Driver
* Push Buttons (for input control)
* DC Motors (for testing)
* Power Supply (Battery / Adapter)

---

## ⚙️ Features

* Compact PCB design
* Motor control using TB6612FNG
* Manual input using push buttons
* Optimized routing for minimal noise
* Beginner-friendly design approach

---

## 🖥️ Software Used

* Altium Designer
* Arduino IDE

---

## 🔌 Circuit Description

* The **Arduino Nano** acts as the main controller
* The **TB6612FNG driver** controls motor direction and speed
* **Push buttons** are connected to digital pins for user input
* Power is regulated and distributed across the PCB

---

## 🧠 Working Principle

1. Push button input is read by Arduino Nano
2. Based on input, control signals are sent to TB6612FNG
3. Motor driver drives the motors accordingly
4. System responds in real-time to user commands

---

## 🧩 PCB Design Process

1. Created schematic with all components
2. Assigned footprints to each component
3. Converted schematic to PCB layout
4. Placed components logically
5. Routed connections (tracks)
6. Performed Design Rule Check (DRC)
7. Generated Gerber files

---

## ▶️ How to Use

1. Upload Arduino code using Arduino IDE
2. Connect power supply
3. Use push buttons to control motor behavior
4. Observe motor response
